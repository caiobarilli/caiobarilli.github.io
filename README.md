### Pré-requisitos
#
Para começar, certifique-se de ter o [NodeJS](https://nodejs.org/en/) no seu sistema.

<br />

### Download
#
Faça o download do projeto com o comando:

```sh
git clone git@github.com:caiobarilli/reactjs-boilerplate.git
```

### Instalação

```sh
yarn
```

```sh
yarn prepare
```

### Reactjs
#

```sh
yarn build
```

```sh
yarn start
```

```sh
yarn lint
```

```sh
yarn test
```

```sh
yarn test -- --coverage
```

### Storybook
#
```sh
yarn storybook
```

```sh
yarn build-storybook
```

### PlopJS
#

```sh
yarn generate ComponentName
```
